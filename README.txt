A customized archive widget with an accordion style layout.

== Description ==

Archive widget sorted by year and month with optional title field, posts counts per year/month and possibility to change animation duration.

**Archive Widget Options: <BR>**
- custom widget title, <BR>
- animation duration, <BR>
- display posts count <BR>
- multilanguage support <BR>



== Installation ==

1. Upload `archive-akkordion.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

You will find the new widget under Appearance > Widgets
